## Backend

The backend is made in flask-restful.

### Running Tests

Tests are done in pyresttest. Run **TESTING.sh**.
Alternatively you can start app.py and run the following command
```
pyresttest http://127.0.0.1:5000 tests/auth.yaml 
```
